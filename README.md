Serviços Users-manager

O objetivo desse repositorio é rodar toda infra-estrutura necessária para a aplicação "users-manager" rodar https://github.com/Joaoasbittencourt/users-manager.

## Como rodar:

1. Certifique-se de ter docker instalado
2. Clone https://github.com/Joaoasbittencourt/users-manager na raiz do projeto
3. Clone https://github.com/Joaoasbittencourt/user-service na raiz do projeto
4. Clone https://github.com/Joaoasbittencourt/fake-store-service na raiz do projeto
5. Execute: `docker compose --profile default up --build`
